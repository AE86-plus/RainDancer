# RainDancer

This repository bundles two related open-source projects:

- `RainDancer/`
- `RainDancer-s/`

## Paper

- [**RainDancer: RGB-Event Video Deraining with Rain-Oriented Spiking Dynamics**](https://arxiv.org/abs/2607.13802)

## Included Projects

- `RainDancer/`: full model
- `RainDancer-s/`: lightweight model

Each subdirectory contains its own README with:

- dataset download information
- pretrained weight download information
- training commands
- inference commands

## Notes

- Both projects use the same released dataset package.
- The pretrained weights are provided separately for `RainDancer/` and `RainDancer-s/`.
- Please read the README inside each subproject before training or testing.
